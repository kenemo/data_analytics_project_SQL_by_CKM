# data_analytics_project_SQL_by_CKM
This is a data analytic project done using SQL. This project was achieved in conjunction with **Baraa Khatib Salkini**, also known as **Data With Baraa**, an IT professional and passionate YouTuber.
The comprehensive collection of SQL scripts used in this project covers various analyses such as database exploration, measures and metrics, time-based trends, cumulative analytics, segmentation, and more data topics.
"This repository contains SQL queries designed to help data analysts and BI professionals quickly explore, segment, and analyze data within a relational database. Each script focuses on a specific analytical theme and demonstrates best practices for SQL queries.", Baraa Khatib Salkini.
